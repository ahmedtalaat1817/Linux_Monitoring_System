# Standardized Output Format
#
# METRIC_NAME|VALUE|STATUS|TIMESTAMP
#
# Examples:
# CPU_USAGE|72.5|WARNING|2026-04-30 15:30:00
# MEMORY_USAGE|45.2|OK|2026-04-30 15:30:00
# DISK_ROOT|92.1|CRITICAL|2026-04-30 15:30:00
# SSHD_SERVICE|active|OK|2026-04-30 15:30:00
# FAILED_SSH|3|WARNING|2026-04-30 15:30:00
#
# STATUS values: OK, WARNING, CRITICAL
